# External Reference Note / 外部参照メモ

## Purpose / 目的

This note is a one-page reference for people who encounter LUMINA-30 from outside the repository network.

It explains how to refer to LUMINA-30 without treating it as an endorsement, mandate, certification, or institutional position.

本メモは、repoネットワーク外からLUMINA-30に到達した人向けの1ページ参照である。

## Recommended External Description / 推奨される外部説明

EN:
LUMINA-30 is a non-binding boundary-check framework for AI governance and incident review. Its core question is whether Human Refusal Authority remained effective before Irreversible Impact. It does not replace alignment research, law, regulation, or institutional review; it provides a compact boundary question that may be used when ordinary oversight language does not show whether refusal was practically available.

JP:
LUMINA-30は、AIガバナンスおよびインシデントレビューのための非拘束的な境界判定フレームである。中核問いは、不可逆的影響の前に人間の拒否権が実効的に残っていたかである。アライメント研究、法律、規制、制度的レビューを置き換えるものではなく、通常の監督表現だけでは拒否が実際に可能だったかを示せない場合に使用できる、簡潔な境界問いを提供する。

## Safe External Use / 安全な外部利用

Safe wording:

- “can be examined through”
- “provides a non-binding boundary question”
- “may be used as a supplementary review prompt”
- “focuses on effective human refusal before irreversibility”

Avoid wording:

- “certifies”
- “approves”
- “requires”
- “proves safety”
- “is adopted by”
- “is the official standard for”

安全な表現：

- 「〜によって検討できる」
- 「非拘束的な境界問いを提供する」
- 「補助的レビュー・プロンプトとして使用できる」
- 「不可逆化前の実効的人間拒否に焦点を当てる」

避ける表現：

- 「認証する」
- 「承認する」
- 「義務づける」
- 「安全性を証明する」
- 「採用されている」
- 「公式標準である」
